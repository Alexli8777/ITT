# 📊 Excel 合併工具

這是一個 GitHub Pages 網頁工具，使用者可以上傳一個包含三個分頁的 Excel 檔案，系統會自動合併並提供下載。

## 功能特色

- ✅ 支援 `.xlsx` 檔案格式
- ✅ 使用 SheetJS 處理 Excel
- ✅ 保留原始三個分頁
- ✅ 前端處理，無需後端伺服器

## 使用方式

1. 開啟 `index.html`
2. 上傳 Excel 檔案
3. 點擊「開始處理」
4. 下載合併後的 Excel 檔案

## 技術架構

- HTML / CSS / JavaScript
- SheetJS (`xlsx.full.min.js`)
